# 🎓 Grade Calculator (Tkinter GUI)

A simple and user-friendly **Grade Calculator** built using **Python and Tkinter**.
This application allows users to enter grades and instantly get the average and corresponding letter grade.

## ✨ Features

* 📊 Calculate average of grades
* 🎓 Automatic grade assignment (A–F)
* 🖥️ Clean and colorful GUI
* 🔢 Accepts 1 to 5 grades
* ⚠️ Error handling for invalid input
* 🔄 Reset functionality

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI library)

## 📥 Input Format

* Enter grades separated by commas
  **Example:**

```
85, 90, 78
```

## 🧠 Grade Criteria

| Average Score | Grade |
| ------------- | ----- |
| 90+           | A     |
| 80–89         | B     |
| 70–79         | C     |
| 60–69         | D     |
| Below 60      | F     |

## ⚠️ Notes

* Only 1 to 5 grades are allowed
* Invalid input will show an error message
